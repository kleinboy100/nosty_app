# KasiConnect

## What this app does
- Connect Takeaways and restaurants to customers through digital inclusion for township economic development and convenience.
- Key user roles (customer / restaurant owner / admin)

## Tech stack
- Lovable (frontend)
- Supabase (Postgres + Auth + RLS)
- Supabase Edge Functions (server-side API layer)
- Yoco Checkout API + Webhooks (payments)
